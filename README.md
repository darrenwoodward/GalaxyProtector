## Key Features

* Component based
  - Custom Godot components 
* Main menu
* Parallax effect background
* Animated character and enemy sprites 
* Difficulty curve based on the number of enemies defeated
  - Random spawning location for enemies
* Highscore 
* Enemy Projectiles
